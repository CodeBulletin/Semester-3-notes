Newton's Interpolation

Newton's interpolation, specifically Newton's divided difference interpolation, is another mathematical method for constructing a polynomial that approximates a function based on a given set of data points. Like Lagrange interpolation, it is used in numerical analysis and mathematics to estimate values between known data points. Newton's interpolation, however, is often favored for its simplicity and computational efficiency.

Newton's divided difference interpolation has some advantages over Lagrange interpolation. It is particularly well-suited for situations where you want to add additional data points without having to completely recompute the interpolating polynomial. You can simply update the divided differences based on the new data points.

While Newton's interpolation is a useful technique, it shares the potential issue of high-degree polynomials with Lagrange interpolation, which can lead to numerical instability for a large number of data points. In practice, it's important to consider the specific characteristics of your data and the desired accuracy when choosing between interpolation methods.

Suppose you have a set of data points \((x_0, y_0),(x_1, y_1), (x_2, y_2), ..., (x_n, y_n)\), where each \((x_i, y_i)\) represents a data point.

Divided Difference

Here's how Newton's divided difference interpolation works:

\(p_n(x) = a_0 + a_1(x - x_0) + a_2(x-x_0)(x - x_1) + ... + a_n\prod_{i=0}^{n-1}(x-x_i)\)

where,
      \(a_0 = f(x_0)\)
      \(a_1 = f(x_0, x_1)\)
      \(a_2 = f(x_0, x_1, x_2)\)
      â€¦.
      \(a_n = f(x_0, x_1, x_2, ..., x_n)\) 

where again,
      \(f(x_i) = y_i\)
      \(f(x_i, x_{i+1}, x_{i+2}, ..., x_{n+1}) = \frac{f(x_{i+1}, x_{i+2}, x_{i+3}, ..., x_{n+1}) - f(x_i, x_{i+1}, x_{i+2}, ..., x_n)}{x_{n+1} - x_i}\)

Forward Difference (Equidistant)(near \(x_0\))

\(p_n(x) = y_0 + \frac{x - x_0}{h^11!}\Delta y_0 + \frac{(x-x_0)(x - x_0 -1)}{h^22!}\Delta^2 y_0 + \frac{(x - x_0)(x-x_0-1)(x-x_0-2)}{h^33!}\Delta^3 y_0 + ... + \frac{\prod_{i=0}^{n-1}(x -x_0 - i)}{h^nn!}\Delta^n y_0\)

where,
      \(\Delta^ky_0\)  is kth forward difference of \(y_0\)
      assume \(u = \frac{x - x_0}{h}\) for easier calculations then \(p_n(x) = y_0 + \sum_{i=1}^{n}\frac{\prod_{j=0}^{i-1}(u - j)}{i!}\Delta^i y_0\)

Backward Difference (Equidistant)(near \(x_n\))

\(p_n(x) = y_n + \frac{x - x_n}{h^11!}\nabla y_n + \frac{(x-x_n)(x - x_n +1)}{h^22!}\nabla^2 y_n + \frac{(x - x_n)(x-x_n+1)(x-x_n+2)}{h^33!}\nabla^3 y_n + ... + \frac{\prod_{i=0}^{n-1}(x -x_n + i)}{h^nn!}\nabla^n y_n\)

where,
      \(\nabla^ky_n\)  is kth backward difference of \(y_n\)
      assume \(u = \frac{x - x_n}{h}\) for easier calculations then \(p_n(x) = y_n + \sum_{i=1}^{n}\frac{\prod_{j=0}^{i-1}(u + j)}{i!}\nabla^i y_n\)
