Gauss Forward and Backward Formula

Forward distance formula

\(p_n(x) = y_0 + \frac{x - x_0}{h^11!}\Delta y_0 + \frac{(x-x_0)(x - x_0 -1)}{h^22!}\Delta^2 y_{-1} + \frac{(x - x_0+1)(x - x_0)(x-x_0-1)}{h^33!}\Delta^3 y_{-1} + \frac{(x - x_0 + 1)(x - x_0)(x-x_0-1)(x-x_0-2)}{h^44!}\Delta^4 y_{-2} + ...\)

or in short let \(u = \frac{x - x_0}{h}\)

\(p_n(u) = y_0 + u\Delta y_{0} + \frac{(u)(u -1)}{2}\Delta^2 y_{-1} + \frac{(u+1)(u)(u-1)}{3!}\Delta^3 y_{-1} + \frac{(u + 1)(u)(u-1)(u-2)}{4!}\Delta^4 y_{-2} + ...\)

Backward distance formula

let \(u = \frac{x - x_0}{h}\)

\(p_n(u) = y_0 + u\Delta y_{-1} + \frac{(u+1)(u)}{2}\Delta^2 y_{-1} + \frac{(u+1)(u)(u-1)}{3!}\Delta^3 y_{-2} + \frac{(u+2)(u + 1)(u)(u-1)}{4!}\Delta^4 y_{-2} + ...\)
