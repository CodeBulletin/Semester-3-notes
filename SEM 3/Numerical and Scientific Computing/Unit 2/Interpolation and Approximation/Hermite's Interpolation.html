Hermite's Interpolation

Hermite's interpolation is an algorithm which generates interpolating polynomials that interpolate values at the given nodes as well as certain derivatives

let \((x_k, y_k, y_k', y_k'', ..., y_k^{(m)})\) for  \(0 \le k \le n\)

we seek polynominal \(f(x)\) such that

\(f(x_k) = y_k, \space 0 \le k \le n\)

and,

\(f^{(j)}(x_k) = y_k^{(j)}, \space 0 \le k \le n, \space 0 \le j \le m\)

\(f(x)\) is the polynomial of degree  \((n + 1)(m+1) - 1\)

case 1: special condition \(m = 0\): use newtons interpolation
case 2: \(m > 0\)

Recall Newtons divided diffrence we add the following definations

\(f[x_i, x_i] = f'(x_i), \space f[x_i, x_i, x_i] = \frac{f''(x)}{2!}, \space f[x_i, x_i, x_i, x_i] = \frac{f'''(x)}{3!}, \space ...\)
