Taylor's Approximation

Given a function \(f(x)\) we can at certain point \(x_0\) can approximate the function using taylors series

** A Maclaurin series is a Taylor series expansion of a function about 0

Taylor's series

\(f(x) = f(c) + f'(c)*(x-c) + f''(c) * \frac{(x - c)^2}{2!} +...+ f^{(n)}(c)*\frac{(x - c)^n}{n!} + ...\)

Approximation of Functions by Taylor Polynomials

Well A Taylor Series goes for ever if we stop it after the  \(n^{th}\) derivative term is called the Taylor polynomial of \(n^{th}\) degree

\(T(x) = f(c) + f'(c)*(x-c) + f''(c) * \frac{(x - c)^2}{2!} +...+ f^{(n)}(c)*\frac{(x - c)^n}{n!}\)

\(f(x) = f(c) + f'(c)*(x-c) + f''(c) * \frac{(x - c)^2}{2!} +...+ f^{(n)}(c)*\frac{(x - c)^n}{n!} + R_n(x)\)

\(f(x) = T(x) + R_n(x)\)

\(R_n(x)\) is the error

\(R_n(x) = \frac{f^{(n+1)}(Z)(x-c)^{n+1}}{(n+1)!}\),  \(Z\) is a number that is between \(x\) and  \(c\)

** note pick the \(Z\) that is maximizes \(R_n(x)\)

\(T(x) \approx f(x)\) at some point  \(c\)
