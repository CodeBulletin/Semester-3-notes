Lagrange's Interpolation

Lagrange interpolation is a method for constructing a polynomial that passes through a set of given data points. It's a technique used in numerical analysis and mathematics to approximate a function when you have a discrete set of data points but want to find values between those points.

The key idea behind Lagrange interpolation is to construct a polynomial of the lowest possible degree that passes through all the given data points. This polynomial is known as the Lagrange polynomial, and it's represented as a weighted sum of basis polynomials.

The Lagrange interpolation formula finds the lowest order polynominal L(x) which interpolates the points (x_i, y_i)

Formula: 

Suppose you have a set of data points \((x_1, y_1), (x_2, y_2), ..., (x_n, y_n)\), where each \((x_i, y_i)\) represents a data point.

The Lagrange's Polynominal is given by: \(L(x) = \sum_{i=0}^ny_il_i(x)\)

where \(l_i(x) = \prod_{j=0; j \ne i}^{n}\frac{x-x_j}{x_i-x_j}\)
