Multi Linear Regression

Multiple regression is an extension of simple linear regression that involves modeling the relationship between a dependent variable and two or more independent variables. In other words, it allows for the consideration of multiple predictors in predicting an outcome. The general form of a multiple regression model with \(k\) independent variables (predictors) can be expressed as:

\(Y = ε + b_0 + b_1x_1 + b_2x_2 + ... + b_kx_k\)

Here:

  • \(Y\) is the dependent variable.
  • \(b_0\)​ is the y-intercept.
  • \(b_1, b_2, ..., b_k\)​ are the coefficients associated with the independent variables \(x_1,x_2, ...,x_k\)​, respectively.
  • \(ε\) is the error term, representing unobserved factors that affect \(Y\) but are not included in the model.

The goal of multiple regression is to estimate the coefficients ( \(b_1, b_2, ..., b_k\)) that best fit the data. This is typically done using the method of least squares, where the sum of the squared differences between the observed and predicted values is minimized.
