False Position Method

The False Position Method, also known as the Regula Falsi Method or the Linear Interpolation Method, is a root-finding algorithm used in numerical analysis to approximate the roots of a continuous function. Similar to the Bisection Method, it operates on an interval where the function changes sign, aiming to refine the estimate of the root with each iteration. However, the False Position Method employs linear interpolation to make a more informed estimate of the root compared to the simple bisection approach.

Here's how the False Position Method works:

  1. Start with an interval [a, b] where the function changes sign. Calculate the function values f(a) and f(b).
  2. Compute the point x that corresponds to the intersection of the straight line connecting the points (a, f(a)) and (b, f(b)) with the x-axis. This point is given by:
    \(x= \frac{af(b) - bf(a)}{f(b) -f(a)}\)
  3. Calculate the function value f(x) at the newly computed point x.
  4. Determine whether the root lies between a and x or between x and b based on the signs of f(a) and f(x).
  5. Update the interval [a, b] to [a, x] or [x, b], depending on the location of the root.
  6. Repeat the process iteratively, calculating new values of x and updating the interval until a desired level of accuracy is achieved.

The False Position Method shares some similarities with the Bisection Method, as both methods work with an interval that contains the root and narrow it down with each iteration. However, the False Position Method typically converges faster than the Bisection Method because it incorporates more information about the function's behavior by using linear interpolation.

Order of Convergence


The Order of convergence of the Regula Falsi (False Position) Method is linear. This means that, similar to the Bisection Method, the error decreases by a constant factor with each iteration. However, compared to the Bisection Method, the linear convergence of the Regula Falsi Method is generally faster, since it uses linear interpolation to estimate the root and incorporates more information about the function's behavior.

In mathematical terms, the Order of convergence can be expressed as follows:

\(|e_{n+1}| = C * |e_n|^k\)

Where:

  • \(e_{n+1}\) is the error in the (n+1)-th iteration.
  • \(e_n\) is the error in the n-th iteration.
  • C is a constant factor less than 1.
  • k is order of Convergence for Regula Falsi Method it is 1

The Regula Falsi Method has a slightly faster convergence rate compared to the Bisection Method because it uses linear interpolation to estimate the root, which takes into account the local behavior of the function.
