General Error Formula

Let \(u = f(x_1, x_2, x_3, ..., x_n)\)

then the error \(\Delta u\) in  \(u\) is

\(\Delta u = \frac{\delta u}{\delta x_1}\Delta x_1 + \frac{\delta u}{\delta x_2}\Delta x_2 + \frac{\delta u}{\delta x_3}\Delta x_3 +... + \frac{\delta u}{\delta x_n}\Delta x_n\)

Then the relative error will be

\(E_r = \frac{\Delta u}{u}\)

Example

Given that \(u = \frac{7xy}{z^3}\) and  \(\Delta x = \Delta y = \Delta z = 0.001\)

take log on both sides

\(log(u) = log(7) + log(x) + log(y) - 3log(z)\)

\((\frac{\delta u}{u})_{max} = |(\frac{1}{x})\delta x| + |(\frac{1}{y})\delta y| + |(\frac{3}{z})\delta z|\)

\((\frac{\Delta u}{u})_{max} = 0.001 + 0.001 + 0.003 = 0.005\)
