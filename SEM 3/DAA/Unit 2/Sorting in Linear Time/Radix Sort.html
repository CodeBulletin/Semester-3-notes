Radix Sort

Radix Sort is a non-comparative integer sorting algorithm that processes the input data by processing individual digits or characters of each element. It works by distributing elements into buckets based on their individual digits, starting from the least significant digit (rightmost) and moving towards the most significant digit (leftmost). After processing each digit position, the elements are collected back in order.

Here's a step-by-step explanation of how Radix Sort works:

  1. Determine the maximum number of digits: Find the maximum number of digits in the input numbers. This determines how many passes the algorithm will make.
  2. Initialize ten buckets (0-9) for each digit position. These buckets are used to temporarily store elements based on the current digit being examined.
  3. For each pass, distribute elements into the buckets: Starting from the least significant digit (rightmost), distribute each element into one of the ten buckets based on the value of the current digit. Elements with the same digit go into the same bucket. After distributing all elements, collect them back from the buckets in the order they were placed.
  4. Repeat step 3 for each digit position, moving from right to left.
  5. After processing all digit positions, the input array will be sorted.

Radix Sort has a time complexity of O(\(O(k * n)\)), where "k" is the number of digits in the maximum number, and "n" is the number of elements in the input array. It is a stable sorting algorithm and is often used for sorting integers or fixed-length strings.

RadixSort(Array, NumDigits)
    1. Let InputArray[1..n] be the array to be sorted.
    2. For each digit i from 1 to NumDigits (least significant digit to most significant digit)
           a. Perform CountSort on InputArray considering digit i.

CountSort(Array, Digit)
    1. Let CountArray[0..9] be an array to store the count of each digit (0-9).
    2. Let OutputArray[1..n] be an array to store the sorted elements.
    3. Initialize CountArray to 0.

    // Count occurrences of each digit
    4. For each element in Array
           Increment CountArray[GetDigit(element, Digit)]

    // Update CountArray to contain actual position of this digit in OutputArray
    5. For i from 1 to 9
           CountArray[i] = CountArray[i] + CountArray[i - 1]

    // Build the output array
    6. For each element in Array, starting from last and moving to first
           a. OutputArray[CountArray[GetDigit(element, Digit)] - 1] = element
           b. Decrement CountArray[GetDigit(element, Digit)]

    // Copy the output array to Array, so that Array now contains sorted numbers according to current digit
    7. Copy OutputArray to Array
