(a) Write the algorithm of insertion sort and then compute its time complexity. 

void insertionSort(int arr[], int size) {
    for (int i = 1; i < size; i++) {
        int key = arr[i];
        int j = i - 1;

        // Move elements greater than the key to the right
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }

        // Place the key in its correct position
        arr[j + 1] = key;
    }
}

Time Complexity

\(T(n) = 1 + 2 + 3 + 4 + 5 + ... + n-1\)

\(T(n) = \frac{n(n-1)}{2} = \frac{n^2}2 - \frac{n}2 = O(n^2)\)

(b) Prove that \(O(n!) = 2^n\)

To prove that \(2^n\) is  \(O(n!)\),  we need to show that \(2^n \le cn!\), for some constant \(c\) and all values of \(n \ge k\), where  \(k\) is also some constant.

now let \(c = 2\), \(k = 1\) we need to prove \(2^n \le 2n! \space \forall \space n\ge1\) 

Proof By Induction:

Step 1: we prove it true for n = 1

\(2^1 = 2 \space and \space 2*1! = 2\) hence  \(2^1 \le 2n!\) for \(n = 1\)

Step 2: assume \(2^n \le 2n!\) for some \(n = p\)
Step 3: now we will prove that \(2^n \le 2n!\) for \(n = p + 1\)

\(2^{(p+1)} = 2 * 2^p\)

and \(2(p+1)! = 2p(p!)\)

since \(p \ge 2\)

\(2 *2^p \le 2p(p!)\)

hence proved
