(a) Write an algorithm to find second minimum in a list of n items. The list is to be scanned only once.

function findSecondMinimum(list):
    min1 = infinity
    min2 = infinity
    for num in list:
        if num < min1:
            min2 = min1
            min1 = num
        else if num < min2:
            min2 = num
    return min2

 

(b) Write an algorithm to store a set in a linked list and then use the algorithm to find union of two sets.

struct Node:
    value
    next

function storeSetInLinkedList(set):
    head = None
    for element in set:
        newNode = Node()
        newNode.value = element
        newNode.next = head
        head = newNode
    return head

function findUnion(list1, list2):
    unionList = None
    current = list1
    while current is not None:
        newNode = Node()
        newNode.value = current.value
        newNode.next = unionList
        unionList = newNode
        current = current.next
    
    current = list2
    while current is not None:
        if not existsInLinkedList(unionList, current.value):
            newNode = Node()
            newNode.value = current.value
            newNode.next = unionList
            unionList = newNode
        current = current.next
    
    return unionList

function existsInLinkedList(head, value):
    current = head
    while current is not None:
        if current.value == value:
            return True
        current = current.next
    return False
