Master theorem

For decreasing function

let \(T(n) = aT(n-b) + f(n)\),    \(a \gt0, b \gt 0\)

Then,

Case 1: if \(a \lt1\):                            \(T(n) = O(f(n))\)

Case 2: if \(a  = 1\):                           \(T(n) = O(n*f(n))\)

Case 2: if \(a \gt 1\):                           \(T(n) = O(f(n)*a^{\frac n b})\)

For dividing Function

let \(T(n) = aT(n/b) + f(n)\),    \(a \ge1, b\gt1, f(n) = O(n^c*log^d(n))\)

(Calculate \(log_b(a)\))

Then,

Case 1: if \(log_b(a) \lt c\):

Case 1.1: if \(d \ge 0\) :                 \(T(n) = O(f(n))\)

Case 1.2: if \(d \lt 0\) :                \(T(n) = O(n^c)\)

Case 2: if \(log_b(a) = c\):

Case 2.1: if \(d \gt -1\) :             \(T(n) = O(n^c*log^{p+1}(n))\)

Case 2.2: if \(d = -1\) :            \(T(n) = O(n^c * log(log(n)))\)

Case 2.3: if \(d \lt -1\) :            \(T(n) = O(n^c)\)

Case 3: if \(log_b(a) \gt c\):                    \(T(n) = O(n^{log_b(a)})\)
