Big Omega (Best Case)

Omega notation describes the lower bound or best-case behavior of a function or algorithm. It provides a lower limit on the growth rate of a function. In other words, it characterizes how a function or algorithm behaves when the input size becomes very large.

For a function \(f(n)\),  \(\Omega(g(n))\) represents an lower bound on its growth. This means that there exists a constant \(c > 0\) and an input size \(n_0\) such that  \(f(n)\) is bounded bellow by \(c *g(n) \space \forall \space n \ge n_0\)
