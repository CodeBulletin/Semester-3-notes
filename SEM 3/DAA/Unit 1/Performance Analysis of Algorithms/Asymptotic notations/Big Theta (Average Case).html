Big Theta (Average Case)

Theta notation provides both upper and lower bounds on the behavior of a function or algorithm. It characterizes the exact growth rate of a function within a specific range.

For a function \(f(n)\),  \(\Theta(g(n))\) represents a tight bound on its growth. This means that there exist constants \(c_1\) and  \(c_2\), along with an input size \(n_0\), such that
\(c_1*g(n) \le f(n) \le c_2*g(n) \space \forall \space n \ge n_0\)

Example: If an algorithm's time complexity is Î˜(n), it means that the algorithm's running time grows linearly with the input size in a specific range.
