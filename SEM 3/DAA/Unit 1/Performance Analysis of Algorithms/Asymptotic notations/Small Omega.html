Small Omega

Small omega notation provides an lower bound that is not tight. It describes a function's growth rate that is strictly greater than another function's growth rate.

For a function \(f(n)\),  \(\omega(g(n))\) represents a condition where \(f(n)\) grows significantly faster than \(g(n)\) as  \(n\) approaches infinity. In other words, the growth of  \(f(n)\) is asymptotically larger than \(g(n)\).

Formally, for any positive constant \(c > 0\), there exists an input size \(n_0\) such that \(f(n)\) is bounded bellow by \(c*g(n) \space \forall \space n \ge n_0\)

Example: If \(f(n) = n^2\) and  \(g(n) = n\), then \(f(n)\) is  \(\omega(g(n))\) because \(n^2\) grows faster than \(n\).
