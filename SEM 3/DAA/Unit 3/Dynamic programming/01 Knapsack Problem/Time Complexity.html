Time Complexity

Time Complexity Analysis:

  1. The Algorithm has two nested loops: one for items \(n\) and one for the weight capacity \(C\)
  2. Each Cell in the table is is filled once total: \(n \times C\)
  3. The Work Done for each cell is \(O(1)\)

There for the time complexity is \(O(nC)*O(1) = O(nC)\)
