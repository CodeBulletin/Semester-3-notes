0/1 Knapsack Problem

The 0/1 Knapsack Problem is a classic optimization problem in computer science and mathematics. It involves selecting a subset of items from a given set, each with its own weight and value, to maximize the total value while keeping the total weight within a given capacity.

Here's a high-level overview of the 0/1 Knapsack Problem:

  1. Initialize a table: Create a table where the rows represent items and the columns represent weight capacities of the knapsack. The size of the table will be (number of items + 1) x (knapsack capacity + 1).
  2. Fill the table: For each item, go through all weight capacities. Calculate the maximum value attainable at each capacity by considering the following two cases:
    • Exclude the item: The value remains the same as it was with the previous item.
    • Include the item: Add the value of the current item to the value obtained by the remaining weight (current weight capacity - item's weight).
  3. Choose the maximum value: For each item and weight capacity, store the maximum value obtained by either including or excluding the current item.
  4. Find the solution: The bottom-right corner of the table will contain the maximum value that can be carried in the knapsack.

\(V[i, j] = max\{V[i-1, j], V[i-1, j-w[i]]+p[i]\}\)
