Huffman Coding

Huffman coding is a lossless data compression algorithm that assigns variable-length codes to characters based on their frequency of occurrence in the input data. It is widely used in various applications, including file compression, data transmission, and encryption.

Here's a step-by-step explanation of how Huffman coding works:

  1. Calculate the frequency of each character in the input data.
  2. Create a priority queue (min-heap) of nodes, where each node represents a character and its frequency.
  3. Initialize the priority queue with the nodes corresponding to the characters and their frequencies.
  4. While there is more than one node in the priority queue:
    • Remove the two nodes with the lowest frequencies from the priority queue.
    • Create a new node with a frequency equal to the sum of the frequencies of the two removed nodes. Make the two removed nodes its left and right children.
    • Insert the new node back into the priority queue.
  5. The remaining node in the priority queue is the root of the Huffman tree.
  6. Traverse the Huffman tree from the root, assigning a '0' to the left child and a '1' to the right child at each step.
  7. Assign the resulting binary codes to the corresponding characters.
  8. Encode the input data using the generated Huffman codes.
  9. Decode the encoded data using the Huffman tree.

\(Time \space Complexity = O(n \space log \space n)\)
