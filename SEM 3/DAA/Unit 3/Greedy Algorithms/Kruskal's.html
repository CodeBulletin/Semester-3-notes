Kruskal's

Kruskal's algorithm is a greedy algorithm used to find the minimum spanning tree (MST) of a connected, weighted graph. The algorithm starts with an empty set of edges and iteratively adds edges in increasing order of their weights, while ensuring that no cycles are formed. When all vertices are connected, the algorithm terminates, and the resulting set of edges forms the MST.

Here's a step-by-step explanation of Kruskal's algorithm:

  1. Sort the edges of the graph in non-decreasing order of their weights.
  2. Create an empty set to hold the MST.
  3. Iterate through the sorted edges. For each edge:
    • If adding the edge to the MST does not create a cycle, i.e., the two endpoints of the edge are not already connected in the MST, add the edge to the MST.
  4. Continue this process until all vertices are connected or there are no more edges to consider.

The algorithm typically uses a disjoint-set data structure to efficiently check for cycles and maintain the connectivity of vertices. The disjoint-set data structure keeps track of disjoint sets of vertices and allows efficient operations such as union and find.

Once the algorithm terminates, the set of edges obtained forms the minimum spanning tree of the graph.
