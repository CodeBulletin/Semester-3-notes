Proof

To prove that 3-CNF-SAT (3-Conjunctive Normal Form Satisfiability) is NP-complete, we follow a similar approach as with CNF-SAT, but with a specific focus on the 3-CNF format. In 3-CNF, each clause in the formula has exactly three literals. The two main steps are:

Show that 3-CNF-SAT is in NP: This involves demonstrating that any solution to a 3-CNF-SAT problem can be verified in polynomial time.

Show that 3-CNF-SAT is NP-hard: This is typically done by reducing a known NP-complete problem (like SAT or CNF-SAT) to 3-CNF-SAT in polynomial time.

1. 3-CNF-SAT is in NP

To prove that 3-CNF-SAT is in NP, we need to demonstrate that given a particular assignment of variables, we can verify whether this assignment satisfies the 3-CNF formula in polynomial time. Since a 3-CNF formula is just a special case of a CNF formula where each clause has exactly three literals, the verification process is similar. We check each clause to see if at least one literal is true. This can be done in polynomial time, hence 3-CNF-SAT is in NP.

2. 3-CNF-SAT is NP-hard

The more challenging part is proving that 3-CNF-SAT is NP-hard. This is done by reducing an already known NP-complete problem to 3-CNF-SAT. The standard approach is to use the SAT problem. We need to show that any instance of SAT can be polynomially transformed into an instance of 3-CNF-SAT that is satisfiable if and only if the original SAT instance is satisfiable.

Here's a sketch of this reduction:

  • Step 1: Start with a Boolean formula from the SAT problem. This formula might be in any form, not necessarily in CNF.
  • Step 2: Convert this formula into CNF using standard logical equivalences.
  • Step 3: Transform the CNF formula into a 3-CNF formula. This is the crucial step.

This transformation is done in such a way that the original formula is satisfiable if and only if the new 3-CNF formula is satisfiable. Moreover, this transformation can be achieved in polynomial time.

By completing these steps, we've shown that any instance of SAT can be reduced to an instance of 3-CNF-SAT in polynomial time, and that verifying a solution to 3-CNF-SAT is possible in polynomial time. Therefore, 3-CNF-SAT is NP-complete.
