Cloud Computing

Cloud computing involves delivering hosted services over the internet and is categorized into IaaS, PaaS, and SaaS. It can be public, offering services to anyone on the internet, or private, serving a limited number with specific access. Cloud computing aims to provide easy, scalable access to computing resources and IT services, with an infrastructure comprising hardware and software components. It's often likened to utility or on-demand computing and is symbolized by a cloud in diagrams to represent the internet.

How it Works:

  • Front End: Includes client devices, browsers, networks, and cloud software applications.
  • Back End: Comprises databases, servers, and computers, functioning as a data repository.
  • Central Server: Manages communication, using protocols and middleware for connectivity between client devices and cloud servers, with dedicated servers for each application.

Benefits:

  1. Scalability: Facilitates easy resource adjustment based on needs.
  2. Cost Savings: Reduces the need for physical infrastructure and operates on a pay-as-you-go basis.
  3. Accessibility: Allows application and data access from anywhere with internet.
  4. Reliability and Availability: Offers robust infrastructure, redundancy, backups, and SLAs.
  5. Security: Implements advanced security, encryption, and regular audits.

Disadvantages:

  1. Internet Dependency: Relies on internet connectivity, with disruptions affecting access.
  2. Limited Control: Users have restricted control over the infrastructure.
  3. Data Security Concerns: Raises issues regarding data privacy, ownership, and compliance.
  4. Potential Downtime: Can experience temporary disruptions.
  5. Vendor Lock-In: Switching providers or migrating data can be complex and costly.

Cybercrimes in Cloud Computing:

  1. Data Breaches: Targeting sensitive data stored in the cloud.
  2. Account Hijacking: Using stolen credentials to misuse cloud services.
  3. Insider Threats: Authorized individuals misusing access for malicious actions.
  4. Attacks on Providers: Targeting cloud service providers to compromise infrastructure.
  5. Data Loss: Due to technical failures, errors, or malicious actions.
  6. Malware and Ransomware: Using cloud services to distribute harmful software.
