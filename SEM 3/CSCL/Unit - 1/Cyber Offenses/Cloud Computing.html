Cloud Computing

Cloud computing is a general term for anything that involves delivering hosted services over the internet. 

These services are divided into three main categories: infrastructure as a service (IaaS), platform as a service (PaaS) and software as a service (SaaS).

A cloud can be private or public. A public cloud sells services to anyone on the internet. A private cloud is a proprietary network or a data centre that supplies hosted services to a limited number of people, with certain access and permissions settings. Private or public, the goal of cloud computing is to provide easy, scalable access to computing resources and IT services.

Cloud infrastructure involves the hardware and software components required for proper implementation of a cloud computing model. Cloud computing can also be thought of as utility computing, or on-demand computing.

The name cloud computing was inspired by the cloud symbol that's often used to represent the internet in flowcharts and diagrams.

Cloud computing works by allowing client devices to access data over the internet, from remote servers, databases and computers.

An internet network connection links the front end (includes the accessing client device, browser, network and cloud software applications) with the back end, which consists of databases, servers and computers).

The back end functions as a repository, storing data that is accessed by the front end.

Communications between the front and back ends are managed by a central server. 

The central server relies on protocols to facilitate the exchange of data.

The central server uses both software and middleware to manage connectivity between different client devices and cloud servers. Typically, there will be a dedicated server for each individual application

 

Benfits of cloud Computing

  1. Scalability: Cloud computing allows businesses to easily scale their resources up or down based on their needs. This flexibility enables organizations to quickly adapt to changing demands without the need for significant infrastructure investments.
  2. Cost Savings: Cloud computing eliminates the need for businesses to invest in and maintain their own physical infrastructure. Instead, they can pay for the resources and services they use on a pay-as-you-go basis. This reduces upfront costs and allows businesses to operate on a more predictable budget.
  3. Accessibility: Cloud computing enables users to access their applications and data from anywhere with an internet connection. This accessibility promotes remote work, collaboration, and flexibility, as users can work from different locations and devices.
  4. Reliability and Availability: Cloud service providers typically offer robust infrastructure and redundancy measures to ensure high availability and reliability. This includes data backups, disaster recovery options, and service level agreements (SLAs) that guarantee a certain level of uptime.
  5. Security: Cloud service providers invest heavily in security measures to protect their infrastructure and customer data. They employ advanced security technologies, encryption, access controls, and regular security audits to ensure data protection.

Disadvantages of Cloud Computing

  1. Dependency on Internet Connectivity: Cloud computing heavily relies on internet connectivity. If there is a disruption in the internet connection, users may experience difficulties accessing their applications or data.
  2. Limited Control: When using cloud services, businesses have limited control over the underlying infrastructure and systems. This can make it challenging to customize or optimize the environment to specific requirements.
  3. Data Security Concerns: While cloud service providers implement robust security measures, some businesses may have concerns about the security and privacy of their data. Storing data on remote servers may raise questions about data ownership, compliance, and potential unauthorized access.
  4. Potential Downtime: Despite high availability measures, cloud services can experience downtime due to technical issues or maintenance. This can temporarily disrupt access to applications and data.
  5. Vendor Lock-In: Moving applications and data to the cloud may create a dependency on a specific cloud service provider. Switching providers or migrating data can be complex and costly, potentially leading to vendor lock-in.

Crimes related to Cloud Computing

Cloud computing has brought numerous benefits to businesses and individuals, but it has also introduced new cybercrime risks. Here are some cybercrimes related to cloud computing:

  1. Data Breaches: Cloud storage and services can be targeted by cybercriminals seeking to gain unauthorized access to sensitive data. This can include personal information, financial records, intellectual property, or trade secrets. Data breaches can occur due to weak security controls, compromised user accounts, or vulnerabilities in the cloud infrastructure.
  2. Account Hijacking: Cybercriminals may attempt to hijack user accounts in cloud services by stealing login credentials through phishing attacks or other means. Once they gain access, they can misuse the account for various purposes, such as data theft, spreading malware, or launching further attacks.
  3. Insider Threats: Insider threats in cloud computing involve individuals with authorized access to cloud resources misusing their privileges for malicious purposes. This can include stealing or leaking sensitive data, sabotaging systems, or causing damage from within the organization.
  4. Cloud Service Provider Attacks: Cybercriminals may target cloud service providers themselves to gain unauthorized access to multiple customer accounts or compromise the underlying infrastructure. This can lead to widespread data breaches or service disruptions affecting multiple organizations.
  5. Data Loss: Cloud storage services are not immune to data loss incidents, whether due to technical failures, human errors, or malicious actions. Data loss can result in the permanent deletion or corruption of important files, causing significant disruptions and potential financial losses.
  6. Malware and Ransomware: Cybercriminals may use cloud services as a distribution platform for malware or ransomware. They can upload infected files or use cloud storage to host malicious software, which can then be spread to other users or systems.
