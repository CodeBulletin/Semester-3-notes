Scree Plot

A Scree Plot is a visual tool in Principal Component Analysis (PCA) to determine the number of principal components to retain. Its key features include:

  1. X-Axis: Represents principal components, ordered by derivation sequence.
  2. Y-Axis: Displays the eigenvalue or variance explained by each component, often as a percentage of total variance.
  3. Appearance: Begins with the first, most variance-explaining component, descending in a manner resembling a scree slope.
  4. Elbow Criterion: Identifies a point where variance decrease shifts from sharp to gradual, indicating significant components.

Purpose:

  • The Scree Plot visualizes principal component importance, aiding in deciding how many components to retain to capture most of the original data's variance.
