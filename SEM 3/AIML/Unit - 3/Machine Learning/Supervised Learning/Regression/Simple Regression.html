Simple Regression

Simple regression, also known as simple linear regression, is a type of regression analysis that models the relationship between a single independent variable (X) and a single dependent variable (Y). It assumes a linear relationship between the variables, meaning that the relationship can be represented by a straight line.

The formula for simple regression can be expressed as:

\(Y = \beta_0 + \beta_1X + \varepsilon\)

Where:

  • \(Y\) represents the dependent variable (the variable being predicted or explained).
  • \(X\) represents the independent variable (the variable used to predict or explain the dependent variable).
  • \(\beta_0\) represents the y-intercept, which is the value of Y when X is equal to 0.
  • \(\beta_1\) represents the slope of the line, which indicates the change in Y for a unit change in X.
  • \(\varepsilon\) represents the error term, which accounts for the variability or noise in the relationship that is not explained by the model.

The goal of simple regression is to estimate the values of  \(\beta_0\) and \(\beta_1\) that best fit the data and minimize the sum of squared errors between the predicted values and the actual values of the dependent variable.

Once the regression model is fitted, it can be used to make predictions for new values of \(X\) by plugging them into the equation and calculating the corresponding predicted values of Y.

It's important to note that simple regression assumes certain assumptions, such as linearity, independence of errors, and constant variance of errors. These assumptions should be checked and validated before interpreting the results of a simple regression analysis.
