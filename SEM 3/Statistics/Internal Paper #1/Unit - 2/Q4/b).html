Consider a binomial experiment with n = 10 and p = 0.10

(I) Compute f(0)

\(f(0) =  {10 \choose 0}p^0(1-p)^{10} = 1 * 0.9^{10} =  0.3486\)

(II) Compute f(2)

\(f(2) =  {10 \choose 2}p^2(1-p)^{8} = 45 * 0.01*0.43 = 0.1935\)

(III) P(x ≤ 2)

\(P(x \le 2) = f(0) + f(1) + f(2) = 0.3486 + {10 \choose 1}p(1-p)^9+0.1935 = 0.3486 + 0.3874+0.1935 = 0.9295\)

(IV) P(X ≥ 1)

\(P(x >=1) = 1 -P(x=0) = 1 - 0.3486 = 0.6514\)

(V) E(x), Var(x) and Sigma

\(E(x) = n * p = 1\)
\(Var(x) = n * p (1- p) = 0.9\)
\(\sigma = \sqrt{Var(x)} = \sqrt{0.9} =0.9487\)
