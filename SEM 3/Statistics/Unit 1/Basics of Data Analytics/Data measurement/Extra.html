Extra

Nominal, ordinal, interval, and ratio are four types of data measurements:

  1. Nominal Measurement: Categorizes data into distinct, unordered groups. Examples: gender (male/female), colors (red/blue/green), product categories (electronics/clothing/books).
  2. Ordinal Measurement: Sorts data into ordered categories, but the intervals between categories are not equal. Examples: rating scales (1-5 stars), education levels (elementary to college), customer satisfaction (very dissatisfied to very satisfied).
  3. Interval Measurement: Assigns numerical values with meaningful, consistent differences, but lacks a true zero point. Examples: temperature (Celsius/Fahrenheit), calendar dates, IQ scores.
  4. Ratio Measurement: Similar to interval, but includes a true zero point, allowing meaningful ratio comparisons. Examples: weight, height, time, income.
