Types of Variables

Categorical Variables

Categorical variables are variables that represent distinct categories or groups and can take on a limited, fixed number of values. These values don't have any inherent numerical meaning. There are two main types of categorical variables:

Nominal Variable: Nominal variables are categorical variables that represent different categories without any inherent order or ranking among them. Examples of nominal variables include colors, gender (male, female, non-binary), and types of animals (dog, cat, bird).

Ordinal Variable: Ordinal variables are categorical variables that have a clear order or ranking among the categories, but the differences between the categories are not necessarily uniform or measurable. Examples of ordinal variables include education levels (high school, college, graduate school), customer satisfaction ratings (e.g., very dissatisfied, dissatisfied, neutral, satisfied, very satisfied), and socioeconomic classes (low, middle, high).

Quantitative Variables

Quantitative variables, also known as numerical variables or continuous variables, represent measurements or quantities that can be expressed as numerical values. These variables can be subjected to mathematical operations and have meaningful numerical interpretations. There are two main types of quantitative variables:

Discrete Variables: Discrete variables are quantitative variables that take on distinct, separate values with gaps between them. These values are typically whole numbers. Examples of discrete variables include the number of cars in a parking lot, the count of students in a classroom, and the number of customer complaints in a week.

Continuous Variables: Continuous variables are quantitative variables that can take on an infinite number of values within a certain range. These values are typically measured using real numbers and can include decimal points. Examples of continuous variables include height, weight, temperature, and time.
