t-statistic

the t-statistic is a measure that helps assess the statistical significance of the difference between the means of two groups. It is often used in hypothesis testing to determine whether there is a significant difference between the sample means or between a sample mean and a known value

\(t_{calc} = \frac{\bar X - \mu}{SE(\bar X)}\)

For Single Mean

\(df = n - 1\)
\(SE(\bar X) = \frac{s}{\sqrt n}\)

For Two Mean

\(df = n_1 + n_2  - 2\)

  1. when \(s_1 = s_2\) (Pooled)
    \(s_p^2 = \frac{(n_1-1)s_1^2+(n_2-1)s_2^2}{df}\)
    \(SE_p(\bar X) = s_p\sqrt{\frac1{n_1}+\frac1{n_2}}\)
  2. when \(s_1 \ne s_2\) (Unpooled / Welch)
    \(SE_w(\bar X) = \sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}\)

Hypothesis testing

Null Hypothesis \(H_0: \mu = k\)

Two Tailed
Alternate Hypothesis \(H_{\alpha} : \mu \ne k\)

using t-critical:
find \(t_{\alpha/2, df}\) from the table

if \(|t| \ge t_{\alpha/2, df}\) then reject null hypothesis
else accept null hypothesis

using p value:
if \(pvalue = 2(p(t \le -|t_{calc}|)) \ge \alpha\) then  accept null hypothesis
else accept null hypothesis

Single Tailed
Alternate Hypothesis \(H_{\alpha} : \mu \gt k\)

using t-critical:
find \(t_{\alpha, df}\) from the table

if \(|t| \ge t_{\alpha, df}\) then reject null hypothesis
else accept null hypothesis

using p value:
if \(pvalue = p(t \le -|t_{calc}|) \ge \alpha\) then  accept null hypothesis
else accept null hypothesis

Confidence Interval

\(\bar X \pm t_{\alpha/2, df} SE(\bar X)\)

Terms

\(t_{calc} = student \space distribution \space value\)
\(\mu = population \space mean\)
\(\sigma = population \space sd\)
\(\bar X = sample \space mean\)
\(s = sample \space sd\)
\(n = sample \space size\)
\(df = degree \space of \space freedom\)
\(sd  = standered \space deviation\)
\(\alpha = level \space of \space significance\)
\(t_{\alpha/2, df} = tcritical \space for \space df, \alpha\)
